### Authentication
<hr>

| Method | Route | Description |
|---------|-------|-------------|
| GET | /sign-up | Display the sign-up page (form) |
| POST | /sign-up | Create a new account  |
| GET | /sign-in | Show the sign-in page |
| POST | /sign-in | Authenticate the user |
| GET | /sign-out | Log the user out |



### Movies
<hr>

| Method | Route | Description |
|---------|-------|-------------|
| GET | /movies | List all movies |
| GET | /movies/:movieId | Movie Details  |
| GET | /movies/new | Display the add new movie page  |
| POST | /movies | Create a new movie  |
| GET | /movies/:movieId/edit | Display the form to edit a movie  |
| PUT | /movies/:movieId | Updates a movie's information  |
| Delete | /movies/:movieId | Delete a movie  |
